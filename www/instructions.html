Welcome to the Arizona State Geological Survey's map download tool. This tool will allow you to select any of our published GIS datasets.

All of our published gisdata currently follow the United State Geological Survey's Geologic Map Schema (GeMS). The GeMs schema recommends releasing data in the proprietary ESRI File Geodatabase format (.gdb), which you can download by selecting ESRI File Geodatabase in the format dropdown menu.

We also provide simplified versions of these geodatabases in the popular ESRI Shapefile, geojson, and KML formats. These simplified versions only contain polygons of the geologic units, contacts, faults, orientaton points, and the unit descriptions. We recommend this simplified version for most users.

The tool will provide you with limited metadata for your selected map (i.e., the year of publication and the map authors), but you can find the full metadata record for the map by following the link to the Arizona Geological Survey document repository that will appear when you select a map.
